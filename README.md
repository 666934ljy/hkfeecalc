# HK Fee Calc

HK Stock Trading Fee Calculator — instantly calculate Hong Kong stock trading fees.

**Live:** [hkfeecalc.com](https://hkfeecalc.com)

## What It Calculates

- Stamp Duty (0.1%)
- SFC Transaction Levy (0.0027%)
- AFRC Levy (0.00015%)
- HKEX Trading Fee (0.00565%)
- CCASS Clearing Fee (0.0042%)
- Broker Commission (configurable)
- Breakeven price and round-trip cost

## Tech

Plain HTML/CSS/JS — zero dependencies, zero build step. Deployed to GitHub Pages.

## Deployment

1. Push to `main` branch
2. GitHub Settings → Pages → Source: `Deploy from a branch` → `main` / `/ (root)`
3. Custom domain: `hkfeecalc.com`
4. DNS: Add CNAME record pointing to `<username>.github.io`
